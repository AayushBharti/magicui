## Usage

```tsx
import { TestimonialWall } from "@/components/magicui/testimonial-wall";
```

```tsx
const testimonials = [
  {
    text: "This component is fantastic for showcasing customer feedback.",
    image: "/path/to/avatar.jpg",
    name: "Jane Doe",
    role: "Marketing Manager",
  },
  // ... more testimonials
];

export function Page() {
  return <TestimonialWall testimonials={testimonials} />;
}
```

## Props

The `Props` table is updated to match your final component.

| Prop           | Type                                                             | Default    | Description                                                |
| -------------- | ---------------------------------------------------------------- | ---------- | ---------------------------------------------------------- |
| `testimonials` | `{ text: string; image: string; name: string; role: string; }[]` | `[]`       | An array of testimonial objects to display.                |
| `className`    | `string`                                                         | `-`        | Additional class names to style the component's container. |
| `direction`    | `'up' \| 'down'`                                                 | `'up'`     | The direction of the scroll.                               |
| `duration`     | `number`                                                         | `20`       | The duration of one full scrolling animation in seconds.   |
| `gap`          | `string`                                                         | `'1.5rem'` | The gap between items (e.g., '1.5rem', '24px').            |
